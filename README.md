# 语雀知识库迁移 Skill

> 将语雀知识库内容复制整理到另一个知识库 —— 清洗+分类合并、去重、拆大文档、逐篇挂目录、断点续传。

**核心理念：复制不搬。原库完全不动。**

**作者：[yehuoshun](https://github.com/yehuoshun)**

**v4 迁移即分类版**：LLM 清洗+分类合并为一次调用，逐篇处理立即挂目录，多分类自动复制文档。

## 功能特性

| 功能 | 说明 |
|------|------|
| 🔄 **跨库复制** | 源库毫发无伤 |
| 🧹 **清洗+分类合并** | 一次 LLM 调用完成清洗和分类 |
| ✂️ **大文档拆分** | >50000 字自动按结构拆分，代码块安全切分 |
| 📂 **逐篇挂目录** | 创建即分类即挂载，不攒到后置阶段 |
| 📋 **多分类复制** | 文档跨越多个分类时自动复制副本 |
| 💾 **断点续传** | `toc_map` 保留已建目录，续传直接复用 |
| 🚦 **限流保护** | 429 → 检查 X-RateLimit-Remaining，=0 等整点 |

## 工具脚本

| 脚本 | 说明 |
|------|------|
| `scripts/migrate.py` | 通用迁移脚本（v4），清洗+分类+创建+挂目录一气呵成 |
